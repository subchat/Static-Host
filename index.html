Subdomain Takeover Demonstration

This page is intentionally hosted to demonstrate a subdomain takeover risk. The hostname was pointed via CNAME to an unclaimed Azure Front Door endpoint (*.azurefd.net), allowing control of content under this domain.

Impact: An attacker could host arbitrary content (e.g., phishing, malware, credential theft) on a trusted subdomain. This PoC is non-malicious and for coordinated disclosure only.

Author: Subham Chatterjee Report to Microsoft (MSRC) About Azure Front Door
If you are the owner of this domain, please remove the dangling CNAME or claim the referenced Azure resource.
